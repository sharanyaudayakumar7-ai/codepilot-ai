# CodePilot AI

AI-powered GitHub Repository Analysis Platform built with FastAPI, React, and OpenRouter.

🚀 Live Demo:
https://your-vercel-url.vercel.app

## Features
- GitHub Repository Analysis
- AI Architecture Detection
- Tech Stack Identification
- Repository Summarization
- FastAPI Backend
- React Frontend

## Tech Stack

### Frontend

* React
* Vite
* Axios

### Backend

* FastAPI
* Python
* GitPython

### AI

* OpenRouter
* DeepSeek Chat

## How It Works

1. User enters a GitHub repository URL
2. Repository is cloned locally
3. README and project structure are analyzed
4. AI generates structured insights
5. Results are displayed in the dashboard
